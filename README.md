# Inge--Software
Proyecto


Este proyecto nace de una idea muy sencilla: queremos ayudar a los adultos mayores a mantenerse activos, entretenidos y, al mismo tiempo, a ejercitar su mente y su cuerpo de una forma divertida. Para lograrlo, vamos a crear un juego especialmente diseñado para ellos. No será un juego cualquiera. Estará pensado para que practiquen dos habilidades muy importantes en el día a día: El reconocimiento de gestos, que es la habilidad para entender y repetir movimientos con las manos o el cuerpo. Algo tan cotidiano como saludar, señalar o seguir una indicación. Pero no queremos que solo jueguen. Queremos que ustedes, como familiares o cuidadores, puedan ver si realmente el juego les está ayudando. Por eso, la herramienta también llevará un registro muy sencillo de sus estadísticas y su progreso: Cuánto han mejorado con el tiempo. En qué juegos les va mejor. Si van necesitando más práctica en algún movimiento o en algún reto de ubicación. De esta manera, ustedes podrán saber, de un vistazo, si los adultos mayores están avanzando, sintiéndose más seguros y divirtiéndose al mismo tiempo. Todo presentado de forma clara, sin números complicados ni pantallas confusas. En resumen, no les entregamos solo un juego. Les entregamos una herramienta amigable, útil y cercana para que los adultos mayores ejerciten habilidades cotidianas mientras ustedes pueden celebrar con ellos cada pequeño logro.
